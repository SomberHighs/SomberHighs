# SomberHighs Portfolio Pro

A polished static portfolio site for Benjamin Evans / SomberHighs. It is designed for GitHub Pages and focuses on professional positioning around Python engineering, backend systems, audio tools, SDR, and robotics.

## Files

- `index.html` - semantic portfolio layout with project cards and contact CTA
- `styles.css` - responsive dark UI with modern cards, typography, and mobile nav
- `app.js` - menu behavior, current year, and scroll reveal animation

## Deploy on GitHub Pages

1. Copy these files into the root of `SomberHighs/SomberHighs` or a dedicated portfolio repo.
2. In GitHub, open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select `main` and `/root`.
5. Save.
